1. Dicee Challenge Step 1 - Create an External Javascript File
    Created a new Javascript file called index.js and link to it from the index.html file. 

2.Dicee Challenge Step 2 - Add Dice Images
    In the Dicee project, there is a folder called images, add the images of dice6.png as the source to both of the <img> elements.

3. Dicee Challenge Step 3 - Create a Random Number
      Inside index.js, create a new variable called randomNumber1 then set the value of this variable to a random number between 1 and 6.

4. Dicee Challenge Step 4 - Change the <img> to a Random Dice
        Use the random number created in the last step to pick out a random dice image between dice1.png to dice 6.png then place this image inside the left <img> element.
        used DOM and the method setAttribute() 
        
5.Dicee Challenge Step 5 - Change both <img> Elements
      Do the same for the right hand side image element. 

6.Dicee Challenge Step 6 - Change the Title to Display a Winner
    Change the text in the h1, (which currently says Refresh Me) to show which user won or if there was a draw depending on the dice values of player 1 (left) and player 2 (right).
    used an if statement, querySelector() and the innerHTML property to change the text in the h1.
